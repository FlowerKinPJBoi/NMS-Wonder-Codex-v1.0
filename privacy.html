PRIVACY-SAFE FIRST-PARTY ANALYTICS

Useful signals.
No explorer dossier.

Wonder Codex measures which public pages and tools are useful so the project can improve. The system is operated by Wonder Codex and deliberately avoids invasive visitor tracking.

What is measured

Anonymous visits, public page routes, record or asset IDs opened, feature actions, broad referring domains, and coarse categories such as desktop/mobile, browser family, and operating-system family.

What is never stored

  • Raw IP addresses or raw browser user-agent strings
  • Cookies, account identities, uploaded filenames, or local save paths
  • Search text, contributor form text, administrator activity, or admin routes
  • Raw save data—the Importer continues to process saves locally

Questionnaire answers

The feedback questionnaire is a deliberate submission, separate from anonymous analytics. It stores your ratings, written answers, pricing and credit preferences, and an optional community name so PJ can plan future improvements. It does not collect payment information, raw IP addresses, raw browser details, uploaded files, passwords, save data, or account identifiers. Responses are visible only through PJ’s owner-protected feedback console.

Retention and control

Anonymous journey details expire after 90 days. Daily aggregate counts remain so long-term project growth can be understood. Browsers sending Do Not Track or Global Privacy Control are not measured.

Who can see it

The analytics dashboard is restricted to the Wonder Codex owner. Contributor-test accounts, application testers, and the public cannot access it.
