# Deploy Wonder Codex Private Apps v1.8.5

Deploy the complete source bundle to the existing Wonder Codex GitHub repository, then rebuild both DigitalOcean components.

No environment-variable or database change is required.

After deployment:

1. Hard-refresh `https://wondercodex.com/admin/apps/`.
2. Unlock with the existing named PJ or Boots credential.
3. If Spaces cannot be inspected, copy the warning displayed inside the unlocked vault.
4. Confirm both app cards and their testing briefs render.
5. Upload the reviewed inner build ZIPs only after the vault reports that private storage is online.
