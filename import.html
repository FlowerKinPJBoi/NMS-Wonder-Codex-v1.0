WONDER SAVE FINDER — v1.5 ALPHA

Select a Steam/GOG or Xbox/Game Pass PC save folder.

Grant read-only access to the folder you choose. Wonder Codex reconstructs save slots and decodes HG data locally in your browser before handing the selected character to the existing Wonder analyzer.

Local decoding
Steam / GOG on Windows

Select the NMS folder or the account folder beginning with st_.

%AppData%\HelloGames\NMS
  1. Click Choose Steam/GOG save folder.
  2. Press Ctrl + L, paste the path above, and press Enter.
  3. Select NMS or the st_… account folder.
Xbox App / Game Pass PC

Select the package folder, SystemAppData, wgs, or the long account folder containing containers.index.

%LOCALAPPDATA%\Packages\HelloGames.NoMansSky_bs190hzg1sesy\SystemAppData\wgs
  1. Click Choose Xbox/Game Pass folder.
  2. Press Ctrl + L, paste the path above, and press Enter.
  3. Select wgs or its long account-ID folder.
Read-only and local: selecting a folder does not upload it. Only normalized Wonder records shown in the preview can be sent to the review queue.
