2. Replace the existing files and commit to `main`.
3. Let both the Static Site and Web Service finish deploying.
4. Confirm `/api/health` reports version `1.3.4` and image storage configured.
5. Open `record.html?id=3849&hotfix=134` and hard-refresh once.

No database migration, Spaces setting, re-upload, or reapproval is required.
