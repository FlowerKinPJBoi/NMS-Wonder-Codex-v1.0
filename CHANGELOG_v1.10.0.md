# Wonder Codex v1.10.0 — Galactic Cluster Map beta

## Added

- Interactive Galactic Cluster Map at `map.html`.
- Galaxy, catalog lane, Wonder type, fauna family, route-evidence, text, and display-mode filters.
- Pan, wheel zoom, cluster inspection, densest-visible-cluster navigation, and heatmap mode.
- Public `GET /map-points` API with privacy-safe, location-backed records.
- Signed portal-coordinate decoding for X/Y/Z, validated against the verified Pegasus vectors.
- Published assets appear only when they have a separately verified acquisition sighting.
- Durable seed/location evidence decision and Wonder Capture Companion roadmap.

## Evidence policy

- Procedural seeds identify specimens; they are never treated as coordinates.
- Wonder locations come from UA/curated routes.
- Asset locations come from separately reviewed sightings.
- Map density is explicitly labeled catalog density, not guaranteed spawn probability.

## Navigation

- Added Map links to the public home, catalog, record, asset, contribution, and import surfaces.
